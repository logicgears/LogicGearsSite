Not only does this mean allowing the web to be used by people with disabilities, but also allowing web pages to be understood by people using browsers other than the usual ones Because a website may go through several teams of designers during its lifetime, it is important that those people are able to comprehend the code and to edit it easily. Writing web pages in accordance with the standards shortens site development time and makes pages easier to maintain.
